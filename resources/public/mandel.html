At your service ...

Hit Display to show the Mandelbrot set, using the values in the parameter fields. The defaults show a good overview.

You can click and drag a zoom box in the display area. Hit Zoom to display the selected area. Back will return you to the previous view.

MandelZoom is based on an article by A. K. Dewdney in "The Armchair Universe".

Source code is available on Github.
